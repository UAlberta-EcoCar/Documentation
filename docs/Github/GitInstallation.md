Before installing Git, you should first create a Github account. See [Creating an Account on GitHub](https://docs.github.com/en/get-started/start-your-journey/creating-an-account-on-github).

# Installing Git

There are a few ways to setup Git on a computer[^1]. 
Theres [GitHub CLI](https://docs.github.com/en/github-cli/github-cli/about-github-cli), [GitHub Desktop](https://github.com/apps/desktop), and the [git CLI](https://git-scm.com/install/windows). 
The git CLI is the most versatile, so I recommend installing that. 

1. Folow the instructions for [installing git](https://git-scm.com/install/) on your operating system.

1. Follow the instructions in [Setting your Git username for every repository on your computer](https://docs.github.com/en/get-started/git-basics/setting-your-username-in-git#setting-your-git-username-for-every-repository-on-your-computer).
    * The username should be the username of your GitHub account.

1. Follow the instructions in [Setting your commit email address on Github](https://docs.github.com/en/account-and-profile/how-tos/email-preferences/setting-your-commit-email-address#setting-your-commit-email-address-on-github)

1. Follow the instructions in [Setting your email address for every repository on your computer](https://docs.github.com/en/account-and-profile/how-tos/email-preferences/setting-your-commit-email-address#setting-your-email-address-for-every-repository-on-your-computer)

You should now be able to use git on your computer.

----

[^1]: These steps are based on [Set up Git](https://docs.github.com/en/get-started/git-basics/set-up-git).
