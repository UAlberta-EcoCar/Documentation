# SMD Capacitor Sizes

From JLCPCB's [blog](https://jlcpcb.com/blog/smd-capacitor-sizes)[^1], default to using size 0603 capacitors. 
For decoupling capacitors, consider going up to size 0805 capacitors.

* **0603 Capacitor Size (Most Common):** The undisputed "sweet spot." It balances small size with ease of handling. It is highly recommended for fast and easy prototype PCB assembly[^1].

* **0805 Capacitor Size:** Very easy to hand-solder. Frequently used for decoupling, higher power circuits, and bulk capacitance[^1].

# SMD Resistor Sizes

From JLCPCB's [blog](https://jlcpcb.com/blog/the-complete-guide-to-smd-resistor-package-sizes)[^2], default to using size 0603 resistors.
For power traces, consider going up to size 0805 resistors or higher.
